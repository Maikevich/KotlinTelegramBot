try — данное ключевое слово используется для отметки начала блока кода, который 
потенциально может привести к ошибке. catch — ключевое слово для отметки начала блока кода, 
предназначенного для перехвата и обработки исключений в случае их возникновения. finally —
ключевое слово для отметки начала блока кода, который является дополнительным. Этот блок 
помещается после последнего блока catch. Управление передаётся в блок finally в любом случае, 
было выброшено исключение или нет.
Общий вид конструкции для обработки исключительной ситуации выглядит следующим 
образом:
try { //код, который потенциально может привести к исключительной ситуации } 
catch(SomeException e ) { //в скобках указывается класс конкретной ожидаемой ошибки 
 //код обработки исключительной ситуации} 
finally { //необязательный блок, код которого выполняется в любом случае}
