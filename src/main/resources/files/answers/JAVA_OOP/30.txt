Статические блоки инициализация используются для выполнения кода, который должен 
выполняться один раз при инициализации класса загрузчиком классов, в момент предшествующий 
созданию объектов этого класса при помощи конструктора. Такой блок (в отличие от нестатических, 
принадлежащих конкретном объекту класса) принадлежит только самому классу (объекту 
метакласса Class).
